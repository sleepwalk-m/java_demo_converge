职位信息

岗位职责：

1、负责产品业务功能开发，能独立完成稍微复杂的功能的需求分析，模块设计，前后端代码编写；

2、直面各种复杂的、创新型的业务场景，并提出优秀的解决方案；


任职要求：

1、本科及以上学历，计算机软件、数学等相关专业，优秀可放宽至大专（学信网）

2、2年及以上JAVA开发的经验,***

3、扎实的Java编程基础，熟悉各种设计模式，理解io、多线程、集合等基础框架，对JVM原理有一定的了解

4、熟练SpringMVC、Spring、mybatis、spring boot、spring cloud等开源框架

5、熟悉mysql或oracle数据库，对sql优化有一定的经验

6、熟悉Tomcat、webLogic、WebSphere，maven,svn,git，IDEA，ecplise 等工具的使用

7、学习能力强，适应能力好，有强烈的责任心，具备耐心细心的品质、积极主动、团队精神强

8、前端有一定的开发经验，熟练Extjs，vue，react，AngularJS,jQuery等前端框架中的其中一种 。

9. 具有金融相关项目开发经验的优先

职能类别：Java开发工程师

关键字：Javavue

公司信息

衡泰软件是中国领先的定量金融软件和服务供应商，致力于帮助客户“信息中洞察机会，机会中实现价值”。公司总部位于杭州，并于北京、上海、深圳等地设有分部。

自2002年以来，衡泰软件基于自身“技术+分析”（Technology + Analytics）的积累，始终聚焦定量金融软件与服务领域，其产品涉及金融市场的前中后交易处理、资产管理、风险管理、定价与投资分析、数据管理与数据服务等领域，新一代产品更是具备跨币种、多资产及直通式处理的能力，全面支持各类现货和衍生品的组合管理与分析。
公司目前团队过千，已为数百家券商、银行、基金、保险、资管、信托等金融机构提供专业的产品和服务，并在多个专注市场取得领先地位。
